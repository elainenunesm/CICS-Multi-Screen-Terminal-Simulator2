📖 Manual do Simulador CICS 3270

🎯 Visão Geral

Este simulador permite testar telas CICS 3270 com navegação, validação de campos e exportação de definições BMS.

📂 1. Carregando Telas

Formatos Suportados:

  • TXT (3270): Arquivos de texto com layout 24x80
    • Use x para campos numéricos
    • Use z para campos alfanuméricos
    • Exemplo: NOME: zzzzzzzzz
  • BMS: Arquivos com DFHMSD, DFHMDI e DFHMDF

Como Carregar:

  1. Clique em "Carregar Telas TXT"
  2. Selecione um ou mais arquivos
  3. Clique em "Carregar Telas"
💡 Dica: Você pode arrastar e soltar arquivos diretamente na área de upload!

🔑 2. Detectando PF Keys

O sistema detecta automaticamente PF keys definidas nos arquivos:

Formato no TXT/BMS:

PF3=SAIR  PF5=LIMPAR  PF7/PF8=NAVEGAR  ENTER=CONFIRMAR

Onde colocar:

  • TXT: Em qualquer linha do arquivo
  • BMS: No atributo INITIAL de um DFHMDF
🔵 Visual: PF keys detectadas aparecem com borda azul na tela

🔄 3. Navegação entre Telas

Criação Automática:

  • Ao carregar telas, regras são criadas automaticamente para PF keys detectadas
  • Você precisa apenas selecionar a tela de destino

Criação Manual:

  1. Clique em "Adicionar Regra de Navegação"
  2. Selecione tela de origem
  3. Escolha a tecla (PF1-PF12, ENTER)
  4. Selecione tela de destino ou ação

Tipos de Ação:

  • 🔀 Navegar: Vai para outra tela
  • 💬 Mensagem: Exibe mensagem na linha 1
🟢 Visual: Regras completas aparecem com borda verde

✅ 4. Validação de Campos

Configurando Validações:

  1. Expanda o painel "Configuração de Validação"
  2. Selecione um campo na lista
  3. Configure validações disponíveis:
    • Campo Obrigatório
    • Tamanho Mínimo/Máximo
    • Não permitir zeros
    • Não permitir espaços
    • Padrões (email, CPF, telefone, etc.)

Teclas de Validação:

Marque quais teclas devem validar os campos antes de executar (padrão: ENTER)

⚠️ Se a validação falhar, a mensagem de erro aparece na linha 1 da tela

📤 5. Exportação

Exportar Regras de Navegação:

  • JSON - Formato estruturado
  • COBOL - Código com EVALUATE
  • SQL - INSERT statements
  • Excel/CSV - Planilhas
  • XML - Integração
  • HTML - Documentação

Exportar Validações:

  • BMS: Definições DFHMDF completas
    • COM variáveis: Cria labels nomeados
    • SEM variáveis: Labels como comentários
  • COBOL: Programa com validações
  • Copybook: Definições de campos
📝 BMS: Textos longos são automaticamente quebrados em múltiplos DFHMDF respeitando as 72 colunas do mainframe

💡 7. Dicas e Truques

  • 🎨 Use o botão 🌓 para alternar entre tema claro e escuro
  • 📥 Importe regras de navegação de projetos anteriores
  • 🔄 Você pode recarregar o mesmo arquivo TXT após modificá-lo
  • 🗑️ Clique no X vermelho para remover telas ou regras
  • 🎯 Campos obrigatórios são validados apenas se marcados
  • 📊 Exporte para Excel para compartilhar com a equipe

❓ 8. Problemas Comuns

PF keys não aparecem:

  • Verifique o formato: PF3=SAIR
  • Certifique-se de que está em uma linha do arquivo
  • Confira o console do navegador (F12) para logs

Arquivo BMS não carrega:

  • Deve conter DFHMSD, DFHMDI ou DFHMDF
  • Verifique a sintaxe do arquivo

Navegação não funciona:

  • Verifique se há regra criada para aquela tecla
  • Certifique-se de que a tela de destino está selecionada

📧 Tem dúvidas? Abra o console do navegador (F12) para ver logs detalhados
