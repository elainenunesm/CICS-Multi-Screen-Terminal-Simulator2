# CICS-Multi-Screen-Terminal-Simulator2

🚀 Criei um simulador de terminal CICS direto no navegador!

Como desenvolvedora Mainframe, sempre senti falta de uma forma mais visual e rápida de prototipar telas antes de ir direto para o COBOL.

Então desenvolvi uma ferramenta que simula um terminal CICS 3270 e ainda gera código automaticamente.

🔗 Teste aqui:
https://elainenunesm.github.io/CICS-Multi-Screen-Terminal-Simulator2/

💡 O que ele faz:

💻 Simula telas estilo CICS 3270

📄 Gera código COBOL + CICS + BMS

🔄 Navegação entre múltiplas telas

📦 Exportação de layouts

📱 Possibilidade de responsividade

🎯 Ideal para prototipação rápida

⚠️ Essa é a versão 2, então ainda pode ter alguns bugs — mas já está funcional e evoluindo.

A ideia é transformar isso em uma ferramenta de apoio para desenvolvedores Mainframe, facilitando a criação e organização de telas.

Se você trabalha com COBOL/CICS ou já trabalhou com telas BMS, me conta o que achou 👇
